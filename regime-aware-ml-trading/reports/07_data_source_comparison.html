Objective¶

Compare two data sources for SPY daily OHLCV data:

  1. yfinance CSV — our primary source, cached in data/raw/spy.csv
  2. Alpha Vantage — an alternative provider via REST API

The goal is to verify that our primary dataset is consistent with an independent source, identify any discrepancies in price or volume, and check whether pattern detection results are sensitive to the data source.

Why this matters:

  • Data quality is the foundation of any quantitative analysis
  • Vendor differences (rounding, adjusted vs. unadjusted prices, volume reporting) can silently bias results
  • Confirming robustness across sources strengthens thesis conclusions

Note: If the Alpha Vantage API key is not available, this notebook simulates the second source by adding small random noise to the yfinance data. This demonstrates the comparison framework and can be re-run with real Alpha Vantage data when a key is available.

    rng = np.random.default_rng(42)
    df_av = df_yf.copy()

    # Add small price noise (~0.01% std) to OHLC columns
    for col in ['Open', 'High', 'Low', 'Close']:
        noise = rng.normal(loc=0.0, scale=0.0001, size=len(df_av))
        df_av[col] = df_av[col] * (1 + noise)

    # Add volume noise (~0.5% std)
    vol_noise = rng.normal(loc=0.0, scale=0.005, size=len(df_av))
    df_av['Volume'] = (df_av['Volume'] * (1 + vol_noise)).astype(int)

    # Drop a few random dates to simulate coverage differences
    drop_idx = rng.choice(len(df_av), size=5, replace=False)
    df_av = df_av.drop(df_av.index[drop_idx])

    print(f'Simulated source: {len(df_av)} rows, {df_av.index[0].date()} to {df_av.index[-1].date()}')

common_yf = df_yf_scanned.loc[common_idx]

overlap_stats = []
for name, col in pattern_cols.items():
    if col in ['near_support', 'near_resistance', 'has_event']:
        yf_dates = set(common_yf[common_yf[col]].index)
        av_dates = set(common_av[common_av[col]].index)
    else:
        yf_dates = set(common_yf[common_yf[col].notna()].index)
        av_dates = set(common_av[common_av[col].notna()].index)

    both = len(yf_dates & av_dates)
    yf_only = len(yf_dates - av_dates)
    av_only = len(av_dates - yf_dates)
    total_union = len(yf_dates | av_dates)
    jaccard = round(both / total_union * 100, 1) if total_union > 0 else 0.0

    overlap_stats.append({
        'Pattern': name,
        'Both Sources': both,
        'yfinance Only': yf_only,
        f'{source_label} Only': av_only,
        'Jaccard %': jaccard,
    })

overlap_df = pd.DataFrame(overlap_stats)
